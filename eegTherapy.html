The Real-Time EEG Therapy Platform enhances EMDR therapy by integrating EEG data from the Muse S headband into a web-based system. This allows therapists to monitor brainwave activity (stress, relaxation, focus) and adjust stimuli in real-time, improving therapy effectiveness by 30%.

Key Features

1. Real-time EEG monitoring using the Muse S headband.
2. React-based dashboard for live EEG visualization and session controls.
3. Flask backend with a GRU model for EEG data processing.
4. Low-latency (sub-200ms) audio/video streaming for seamless therapy sessions.
5. Supports up to 10 concurrent therapy sessions.

Technologies Used

1. Flask
2. Python
3. React and React-Plotly
4. Muse S and Muse SDK
5. GRU (Gated Recurrent Unit) Model

How It Works

The system captures EEG signals using the Muse S headband and processes them via a GRU model to detect stress, relaxation, and focus levels. The therapist dashboard visualizes this data in real-time, allowing dynamic adjustments to visual/audio stimuli, enhancing therapy outcomes.

Future Enhancements

1. AI-driven insights to suggest optimal therapy adjustments.
2. Expanded therapist dashboard with session history and trend tracking.
3. Remote therapy support with real-time EEG monitoring.
